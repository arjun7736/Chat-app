A real-time chat application built with a full-stack JavaScript stack.  
It comprises a frontend and backend that enable users to join chat rooms, send messages, and see live updates.

## Features
- Real-time messaging capabilities (chat rooms, one-to-one)  
- Frontend built using modern frameworks  
- Clean backend API for managing rooms, messages & users  
- Simple UI/UX for quick deployment & demonstration  
- Structured project: separate `frontend/` and `backend/` modules  

## Technologies Used
- **Frontend**: React ,Tailwind css , Zustand 
- **Backend**: Node.js + Express  
- **Realtime**: WebSockets (e.g., Socket.io)  
- **Database**:  MongoDB   
- Optional: environment config, logging, etc.

## Getting Started

### Prerequisites
- Node.js (v14+ recommended)  
- npm or yarn  
- Database setup  
- Internet connection for real-time features  

### Installation

1. Clone the repository:  
   ```bash
   git clone https://github.com/arjun7736/Chat-app.git
   cd Chat-app
   cd backend
   npm install
   cd ../frontend
   npm install
   ```
2. Run App
   Backend
   ```bash
   cd backend
   npm run start
   ```
   Frontend
   ```bash
   cd frontend
   npm run dev
   ```
